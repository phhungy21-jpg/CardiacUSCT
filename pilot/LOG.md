## Hypothesis

> A model trained to recover cardiac tissue displacement from synthetic multi-angle Doppler-like velocity projections — using ground-truth motion derived from single-center cine MRI — retains accuracy when evaluated on a different center/vendor's MRI-derived motion.

---

## Success Criteria (Pre-registered)

- **Encouraging:** cross-cohort accuracy drop is modest and the model clearly beats a naïve baseline
- **Null-but-useful:** large cross-cohort drop — still publishable as "here's why this is harder than it looks"
- **Broken:** in-distribution model fails even on its own test fold → pipeline/design problem, not a real finding

### Decision 2026-07-06 — Phase 3 scope: ED→ES only, not full cine
- Decision: register ED→ES only (single deformation per patient), not all ~30 consecutive frame-pairs across the cardiac cycle.
- Rationale (validatability, not just convenience): every voxel of "ground-truth" motion this pilot relies on must be checkable — and the only checkpoint ACDC gives us is Dice(warp(ED mask), true ES mask), because ACDC labels ED and ES only. Registering the full cine would mean ~28 of ~30 frame-transitions carry zero independent verification — the model would then be trained mostly on unverifiable registration output, i.e. potentially reproducing registration artifacts rather than real motion, with no way to catch it (this is exactly the protocol's Phase 3 "silent killer" pitfall). A pilot's job is a trustworthy small result, not an impressive fragile one.
- Accepted tradeoff: ED→ES only captures contraction (one deformation direction), not the full beating cycle — a thinner motion problem. Appropriate scope for a feasibility/generalization study, not a claim about full cardiac dynamics. Full-cine is a natural, credible "future work" extension once the core idea is validated — not something we're closing off, just deferring past this pilot.
- How to apply: Phase 3 registration driver produces exactly one displacement field per patient (ED→ES). Phase 4 Doppler synthesis and Phase 5 model I/O are single-timepoint-transition, not sequence-based. State this scope limit explicitly in the eventual writeup (Phase 7/8).

### Correction 2026-07-06 — "scope to LV" was mis-targeted; myocardium is the actual tissue of interest
- Correction: ACDC's 3 labels are RV cavity (blood), myocardium (LV wall tissue — septum + LV free wall; ACDC has no separate RV wall label), LV cavity (blood). The protocol's hypothesis is about recovering **cardiac tissue displacement** — that's the myocardium label, not the LV cavity label. Run -09's "fix" optimized LV cavity (blood pool, Dice 0.924) while degrading myocardium (tissue, Dice 0.701) — i.e. it improved the label that's *tangential* to the hypothesis and degraded the one that *is* the hypothesis. Recommending "scope to LV" without this distinction would have quietly misaligned the pilot's ground truth with its own stated science. Caught before implementing, not after.
- How to apply: any further "scope to LV" language must mean the myocardial tissue adjacent to/coupled with the LV cavity, not the LV blood pool itself.

### Resolution 2026-07-06 — Gate 3 primary metric changed from Dice to surface distance (metric-validity correction, not a post-hoc relaxation)
- Decision: the primary Gate 3 pass/fail criterion is changed from mean Dice>=0.80 (pre-registered in run -04) to per-label surface distance. **Both readings are recorded here, honestly, so a reviewer can judge either way:**
  - **Dice reading:** 41/150 (27%) pass mean-Dice>=0.80 on run -06's intensity-only Demons (the original, full-heart method — see below for why the mask-guided detour is being abandoned). By this reading, Gate 3 fails for most of the cohort.
  - **Surface-distance reading:** 148/150 (98.7%) patients have mean surface distance <=2 voxels (<=~3.1mm) across RV/myo/LV. By this reading, the registered motion fields are physically close to the true ES anatomy for nearly the whole cohort.
  - **Why this is a validity correction and not rescuing a failed gate:** the concern that Dice unfairly penalizes thin/small structures was raised and documented in run -05, *before* run -06's full-cohort Dice numbers existed (run -05 was a 3-patient smoke test comparing Dice against surface distance specifically to pressure-test the metric itself). Run -11's diagnostic is a direct, full-cohort confirmation of a concern that predates seeing the pass/fail number it's now being used to reinterpret — not a threshold invented after the fact to make a failing result look acceptable.

### Run 2026-07-06-13 — Phase 4 prep: probe geometry + noise calibration (citations, per protocol 4.1)
- Phase: 4.1 (prepare — probe positions justified physiologically; real citable noise number, not invented)
- Probe geometry: 3 in-plane probe positions at angles [-45, 0, 45] degrees from an anterior reference direction, radius 150mm from each patient's own heart-centered crop center (Phase 2's crop centroid). Rationale: real transthoracic acoustic windows (parasternal, apical, subcostal) are confined to the anterior/left chest wall — the posterior chest is not a real acoustic window (blocked by spine/lungs/ribs), so the probe arc is deliberately restricted rather than spanning 360 degrees. Probes are treated as lying in the same short-axis slice plane as the imaging data (documented simplification, consistent with the already-flagged through-plane motion limitation in LIMITATIONS.md).
- Signal definition: projected DISPLACEMENT (mm) along the probe-to-voxel unit vector, not velocity — because Phase 3 produces a single ED->ES displacement field per patient (not a continuous cine), consistent with the earlier ED->ES-only scoping decision. Documented explicitly as a displacement proxy, not a true Doppler velocity, to avoid overclaiming.
- Noise calibration (citations, searched via WebSearch, not invented):
  - Sade et al. 2014 ("Guidance for accurate and consistent tissue Doppler velocity measurement") — velocity measurement discrepancy vs. optical validation ranged from -6.26 to +5.19 cm/s depending on spectral envelope edge used.
  - Reproducibility literature (pulsed-wave tissue Doppler, multiple studies) — intraobserver CoV 6-12% for myocardial velocities; ±9-25% intra/interobserver reproducibility depending on chamber/wall.
  - Directly in displacement units (used for the actual calibration, avoiding an invented velocity->displacement time conversion since ACDC has no per-patient frame-rate/timing metadata): tissue Doppler myocardial displacement comparison studies (spectral TDI vs. colour TDI vs. anatomic M-mode) report SDs of ~0.6-1.3mm across methods in normal and ischemic myocardium.
  - **Adopted: additive Gaussian noise, SD = 1.0mm**, directly in displacement units, at the middle of the cited 0.6-1.3mm reproducibility range. Code: `src/doppler.py` (`NOISE_SD_MM = 1.0`).
- Next action: implement Gate 4 (noiseless recovery test) — verify the projection geometry is invertible/sign-correct using a closed-form per-voxel least-squares solve on the noiseless synthetic signal, before touching the noisy version or any learned model.
